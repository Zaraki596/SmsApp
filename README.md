# SmsApp

A sms app where it reads the local system sms file and notifies whenever their is new sms.

## About

* Shows list of sms from the system.
* Sms are sorted on basis of 1, 2, 3, 6, 12 hours and so on.
* Recieves the notification whenever new sms has been sent.
